# Atividades do Curso de Python - DIO / Luizalabs

Este repositório reúne as atividades, desafios e projetos desenvolvidos durante o
curso de **Python com foco em back-end** da DIO em parceria com o Luizalabs.

O objetivo é manter um histórico organizado da evolução do aprendizado, aplicando
boas práticas desde o início: versionamento com Git, estrutura de pastas,
docstrings, tipagem estática, formatação automática e organização do código.
